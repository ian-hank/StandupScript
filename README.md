### Pretext
StandupScript is a small compiler project I built as a fun side project to maintain my C++ skills and compiler fundamentals. All code in this repository was written by hand without any AI tooling. The only exception to that being any README content below the StandupScript header. 

The official spec for implementation of the StandupScript compiler will be in a separate COMPILER_SPEC.md. You may use this spec as a rigid guideline to implement the compiler for the StandupScript language yourself.

**Unix**
```unix
cmake -S . -B build
cmake --build build

./build/standupc <input.stup>
```
